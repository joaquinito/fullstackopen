# unicafe with Redux

Solutions for exercises 6.1 and 6.2 of the Full Stack Open course.

Link: https://fullstackopen.com/en/part6/flux_architecture_and_redux#exercises-6-1-6-2

Set up with [Vite](https://github.com/vitejs/vite).

## Requirements

* npm

Install dependecies:

```bash
npm install
```

## Run the app

Run the command:
```
npm run dev
```
